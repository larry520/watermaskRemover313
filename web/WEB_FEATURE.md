## 💻 技术实现

### 前端技术
- **HTML5** - 语义化标签
- **CSS3** - 渐变、动画、响应式
- **JavaScript (ES6+)** - 异步处理、事件监听
- **Fetch API** - AJAX 请求

### 后端集成
- **FastAPI** - REST API
- **静态文件服务** - 托管 Web 界面
- **CORS** - 跨域支持
- **文件上传** - Multipart/form-data

### 交互流程

```
用户操作                  前端处理                后端处理
   │                       │                       │
上传文件  ──────────→  FormData  ─────────→  保存文件
   │                       │                       │
点击处理  ──────────→  POST请求  ─────────→  创建任务
   │                       │                       │
   │                   轮询状态  ←─────────  更新状态
   │                       │                       │
   │                   显示进度  ←─────────  处理进度
   │                       │                       │
查看结果  ──────────→  GET请求  ─────────→  返回文件
   │                       │                       │
下载文件  ──────────→  触发下载  ←─────────  文件流
```

## 🔧 配置和部署

### 开发环境

```bash
# 启动开发服务器（自动重载）
uvicorn api.app:app --reload --host 0.0.0.0 --port 8000
```

### 生产环境

```bash
# 使用 gunicorn（多进程）
gunicorn api.app:app -w 4 -k uvicorn.workers.UvicornWorker --bind 0.0.0.0:8000
```

### Docker 部署

```bash
# 构建镜像
docker build -t watermark-remover .

# 运行容器
docker run -p 8000:8000 watermark-remover
```

### 端口配置

修改 `start_web.py` 中的端口：

```python
uvicorn.run(
    app,
    host="0.0.0.0",
    port=8000,  # 修改这里
    log_level="info"
)
```

## ⚙️ 常见问题

### Q1: 如何修改端口？

编辑 `start_web.py`:
```python
uvicorn.run(app, host="0.0.0.0", port=8080)  # 改为 8080
```

### Q2: 如何允许远程访问？

```python
uvicorn.run(app, host="0.0.0.0", port=8000)  # 已经是 0.0.0.0
```

然后访问: http://服务器IP:8000

### Q3: 上传失败？

检查：
- 文件大小限制
- 网络连接
- 服务器日志
