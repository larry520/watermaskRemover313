## 🔧 技术细节

### 前端改动

**index.html**:
- 删除了快速视频标签页
- 删除了质量选择控件
- 删除了跳帧选择控件
- 简化了视频处理选项区域

**app.js**:
- 删除了快速视频相关代码 (~60行)
- 修改视频上传不再发送 quality 和 skip_frames 参数
- 修改批量处理不再发送 quality 参数

### 后端改动

**app.py**:
- 修改 `upload_video()` 接口签名，移除 quality 和 skip_frames 参数
- 修改 `process_video_task()` 使用 skip_frames=1, quality=None
- 修改 `process_fast_video_task()` 使用 quality=None
- 修改 `process_video_frame_by_frame_task()` 使用 skip_frames=1, quality=None

### 参数说明

**quality=None**:
- 自动检测原视频的编码参数
- 使用相同的比特率和质量设置
- 确保输出与原视频质量一致

**skip_frames=1**:
- 处理每一帧，不跳过
- 最高质量输出
- 处理时间最长，但效果最好

## 💡 使用建议

### 普通用户

**图片去水印**:
1. 上传图片
2. 点击"开始处理"
3. 下载结果

**视频去水印**:
1. 上传视频
2. 确认"保留音频"已勾选
3. 点击"开始处理"
4. 下载结果

### 高级用户

**如果需要逐帧检测**:
- 适用于水印位置变化的视频
- 勾选"⭐ 逐帧检测水印"
- 注意：处理时间会较长

**批量处理不同素材**:
- 选择"逐帧检测（不同素材）"模式
- 程序会自动适应每个视频的水印
